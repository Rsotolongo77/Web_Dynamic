# Web_Dynamic

# Web Dynamic is a bootstrapped React app with animations for the company Web Dynamic LLC, A start-up web design company.

## Technologies used:
1. React
2. EmailJs
3. CSS with animations
4. AOS Library

## To run the app simply open terminal and npm start.
